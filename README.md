# 📚 Construction Progressive d'un Graphe de Connaissances

## 🎯 Objectif

Ce projet a pour objectif de découvrir les principes des **graphes de connaissances (Knowledge Graphs)** en construisant progressivement un graphe dans un domaine choisi par les étudiants.

À la fin du projet, les étudiants devront être capables de :

- Identifier les concepts d'un domaine de connaissance.
- Modéliser ces concepts sous forme de nœuds (entités) et de relations.
- Construire un graphe de connaissances simple.
- Développer un programme permettant de générer automatiquement ce graphe.

---

## 👥 Composition des groupes

Le projet est réalisé par des groupes de **3 à 5 étudiants maximum**.

---

# Étapes du projet

## 📝 Étape 1 : Choix du domaine

### Travail demandé

Choisir un domaine de connaissance et rédiger une courte description (5 à 10 lignes) expliquant :

- Pourquoi ce domaine a été choisi.
- Les principaux concepts présents dans ce domaine.
- Les types d'informations pouvant être représentés dans un graphe de connaissances.

### 📦 Livrable

Un document présentant le domaine choisi.

---

## 🧩 Étape 2 : Identification des types de nœuds

### Travail demandé

Identifier les principales entités du domaine.

Chaque type de nœud devra préciser :

- Son nom
- Sa signification
- Les attributs possibles

### 📦 Livrable

Une liste des types de nœuds accompagnée de leurs attributs.

---

## 🔗 Étape 3 : Identification des relations

### Travail demandé

Définir les relations existantes entre les différents types de nœuds.

Pour chaque relation, préciser :

- Le nœud source
- Le nœud cible
- Le nom de la relation
- Sa signification

### 📦 Livrable

Un tableau récapitulatif des relations.

---

## 🚀 Résultat attendu

À l'issue du projet, le dépôt GitHub devra contenir :

- La description du domaine choisi.
- Les types de nœuds et leurs attributs.
- Les relations entre les entités.
- Le code permettant de générer automatiquement le graphe de connaissances.
- Les fichiers du graphe généré (si applicable).

## 🏘️ Domaine choisi : Un quartier
Description

Nous avons choisi comme domaine d'étude un quartier, car il représente un environnement de vie dans lequel de nombreuses personnes, lieux et services interagissent au quotidien. Ce domaine offre une grande diversité d'entités et de relations, ce qui en fait un excellent exemple pour la construction d'un graphe de connaissances.

Un quartier est constitué de plusieurs éléments, notamment des habitants de différents âges (enfants, adolescents, adultes et personnes âgées), des maisons, des écoles, des commerces, des centres de santé, des routes et d'autres infrastructures. Toutes ces entités sont reliées entre elles par différentes interactions, telles que les lieux de résidence, les services fréquentés, les déplacements ou les activités quotidiennes.
